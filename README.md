# SentimentX---Code-Switched-Sentiment-Analysis
Fine-tuning pipeline for mBART-large-50 on code-switched Roman Urdu/English social media text for 3-class sentiment analysis using PyTorch &amp; Hugging Face. Features full fine-tuning vs. PEFT LoRA comparison, 90%+ parameter reduction, back-translation augmentation via MarianMT, and real-world inference.
